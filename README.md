# Task Manager Web App

An interactive task manager built for NTTS Stage 7 Front End Web Development.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (DOM Manipulation)

## Features

- Add tasks with Enter key support
- Mark tasks as complete and toggle back to pending
- Delete individual tasks
- Task counters:
  - Total tasks
  - Pending tasks
  - Completed tasks
- Clear all tasks with confirmation

## How It Works

Tasks are stored in a JavaScript array as the source of truth.

The application follows the flow:

Data → Render → DOM Updates

Whenever a task changes:
1. The array is updated
2. The task list is cleared
3. The updated tasks are displayed again
